# ShowTimeHub

**ShowTimeHub** is a full-stack movie ticket booking application that allows users to browse movies, view show timings, select seats, and book tickets online. The project demonstrates web development using **HTML, CSS, Python, and SQL**.

---

## Features

- Browse and search movies
- View show timings for each movie
- Select seats and book tickets
- Store bookings securely in a SQL database
- Responsive and user-friendly interface

---

## Tech Stack

- **Frontend:** HTML, CSS
- **Backend:** Python (Flask)
- **Database:** SQL (MySQL)
